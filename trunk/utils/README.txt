splitrom - нарезает ПЗУ на 8КБ страницы.

splitdisk - разрезает образ ROM диска на образы ПЗУ. 

Страницы и образы ПЗУ нужно сжать используя утилиту gzip:
gzip -9 *.rom
Получившиеся файлы *.rom.gz переименуте в *.roz, после чего
их можно использовать с эмулятором.

Для создания ROM диска, используйте заготовку в ramdisk.zip.
Примонтируйте его:

mkdir /tmp/ROMDisk
mount -o loop -t msdos ramdisk.img /mnt/ROMDisk

Скопируйте в /mnt/ROMDisk нужные файлы, отмонтируйте

umount /tmp/ROMDisk

и создайте страницы ПЗУ.
